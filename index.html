Match-Match Game

    Game rules:
  • When the game begins, all pictures are hidden.
  • Only 2 cards can be chosen simultaneously.
  • If the cards matches - they will be hidden =)
    To Play:
  1. Select 2 cards to try to match the pictures.
  2. If you match the pictures you can go again.
  3. If they do not match is the computer turn them.
  4. The player that finds all pairs wins!
  5. Have fun this game! =)
